# TwitchChatLogger

## How to use

**Follow this steps**:
</br>*1.*
</br>Open bot.js
</br>Add username, password(TOKEN), channels(if you want more channel: 'channel1', 'channel2').
```sh
identity: {
    username: 'Username',
    password: 'oauth:TOKEN'
},
channels: ['Channel']
```
</br>*2.*
</br>Open Terminal
```sh
npm i express
node server.js
```
</br>*3.*
</br>Open Second Terminal
```sh
node bot.js
```
